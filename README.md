Mostly Env
==========

[![Build Status](https://travis-ci.org/mostlyjs/mostly-env.svg)](https://travis-ci.org/mostlyjs/mostly-env)

Mostly microservice runtime environment module for
* Logging
* Configuration
* Error Reporter
* Redis Sentinel
* Statistics
* Error Hnadling

# Usage

## Installation

```bash
npm install mostly-env
```
